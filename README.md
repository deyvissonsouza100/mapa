# Centro Operacional · GCM BH

Projeto estático para GitHub Pages com:
- cadastro manual de VTR
- busca por endereço
- exibição por data e hora
- importação semiautomática de PDF
- integração opcional com Supabase

## Antes de publicar
1. Edite `config.js`
2. Preencha:
   - `supabase.url`
   - `supabase.anonKey`

## SQL
Rode `sql/schema.sql` no SQL Editor do Supabase.

## GitHub Pages
Publique pela branch `main` na pasta raiz `/`.
